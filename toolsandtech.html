Introduction

GitHub

GitHub is an online repository for constructing computer software, it allows multiple programmers to work, collaborate and access work on their own computer in real time. Changes to code are highlighted and tracked so other users can comment on changes to code. GitHub supports a wide variety of formats and features including Email notifications, nested task lists within files, bug tracking and can also track individuals' contributions, commits, code frequency, with all members activity recorded projects can be easily managed. Git Hub is also open sourced; sharing, and teaching is encouraged. Private repositories are also available and its use for all services is free, including training programs.

Scratch

Scratch is a free programming language provided by MIT that allows users to create interactive stories, animations, and games. The block-style language is a transitional form from pseudo-code to a programming language such as Python. It is a useful teaching aid for beginning programmers to think creatively and develop the systematic reasoning and logic required to become proficient in all computer programming languages https://scratch.mit.edu/
